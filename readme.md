# tickbook
Proyek Laravel dengan Vue.js

## Penggunaan
```bash

# instalasi pustaka Vue
npm install

# production mode Vue
npm run prod

# production mode and watch Vue project
npm run watch

# menjalankan project
php artisan serve

```
